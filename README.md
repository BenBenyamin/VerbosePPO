## Verbose PPO

This project aims to be a one-stop solution for all your Proximal Policy Optimization (PPO) questions, where every nitty-gritty implementation detail is explained in a verbose manner. It is primarily designed for educational purposes, while still being a fully functional PPO implementation. Check out `ppo.py` for more information.


## Project Structure

```
├── example
│   ├── example.py                 # Sample script demonstrating PPO training and evaluation
│   ├── PPO_100_000_CartPoleV1     # Saved model after 100,000 training steps
│   └── PPO_200_000_CartPoleV1     # Saved model after 200,000 training steps
├── images                         # Images for mathematical derivations. Not part of the codebase. 
├── ppo.py                          # Policy Optimization implementation
├── README.md                     
```
